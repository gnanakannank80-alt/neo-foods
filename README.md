# neo-foods
🍔 Food Shop Website

This project is a Food Shop Website developed using HTML, CSS, and JavaScript for the frontend interface, while Google Apps Script is used as the backend service and Google Sheets is used as the database.


The website allows users to view food items, place orders through a form, and store the order details directly into a Google Sheet. This project demonstrates a simple full-stack web application using cloud-based tools without requiring a traditional server.
___________________________________
📌 Project Overview

The main purpose of this project is to build a simple online food ordering website for small restaurants or food businesses.

Customers can browse available food items and submit their order details. Once the order is submitted, the information is sent to a backend service built with Google Apps Script, which automatically saves the order into Google Sheets for easy management.
This project shows how frontend technologies can interact with a cloud backend to store and manage user data.
___________________________________
✨ Key Features

• Responsive food shop website design
• Display food menu items with details
• Simple order form for customers
• Automatic data submission to backend
• Order details stored in Google Sheets
• Lightweight backend using Google Apps Script
• Easy to deploy and manage
___________________________________
🛠 Technologies Used

Frontend

HTML is used to create the structure of the website.
CSS is used to design and style the layout and make the website responsive.
JavaScript is used to handle user interactions and send data to the backend.

Backend

Google Apps Script is used to receive the form data and process the order request.

Database

Google Sheets acts as a database where all customer orders are stored in rows.
___________________________________
⚙️ How the System Works

The user opens the food shop website in the browser.

The user browses the menu and fills in the order form.

The website sends the order details to the backend service.

Google Apps Script receives the request.

The backend stores the order information into Google Sheets.

Each new order is added as a new row in the spreadsheet.
___________________________________
📊 Database Structure
The Google Sheet stores the following information:

• Customer Name
• Phone Number
• Food Item Ordered
• Quantity
• Delivery Address
• Order Time
This allows the shop owner to easily track and manage customer orders.
